Die Alexander Funktion ist eine Invariante für Gruppen, i.e. jeder Gruppe wird eine rationale Funktion zugeordnet. Sind die rationalen Funktionen auch nach Multipikation mit Monomen ungleich, können die Gruppen nicht isomorph sein.
Allerdings hängt diese Invariante zusätzlich von einem gewählten nicht trivialen Homomorphism zu eine freien abelschen Gruppe ab. Für eine gegebene Gruppenpräsentation lässt sich dieser kanonisch wählen. 
Die mathematischen Hintergründe kann man z.B. unter
https://www.ams.org/open-math-notes/omn-view-listing?listingId=111368 -> Knots and links -> The Alexander function of groups
nachlesen.
Die Funktionsweise des Programms wird nach Ausführung via dem Command "help" erklärt.
